CalCentral Live Styles Reference

This page demonstrates all UI widgets and elements currently in use on CalCentral (plus some that are currently unused). Foundation styles are inherited from the currently installed version of Foundation, and overriden in places by CalCentral customizations. Foundation includes more UI elements than are represented here.

The purpose of this style guide is both prescriptive and descriptive, i.e. it shows what comes with Foundation (whether we're currently using it or not), and it demonstrates our significant overrides. In other words, it documents both what we are using and what we can use.
